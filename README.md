# QuantEdge

## Trading Analytics & Risk Monitoring Platform

### Problem Statement
Funded traders often manage multiple trading accounts and strategies but lack a centralized platform to track performance, risk, drawdown, and execution quality.

### Objective
QuantEdge provides a centralized analytics platform that enables traders to monitor account performance, analyze trading behavior, track risk metrics, and generate actionable insights from trading data.

### Key Analytics Metrics
- Daily PnL Analysis
- Win Rate Tracking
- Drawdown Monitoring
- Strategy-wise Performance Analysis
- Account-wise Performance Comparison
- Trade Outcome Distribution (TP, SL, BE, No Result)
